hexoでの記事投稿メモ

hexoで記事を投稿するためのコマンドなど

  1. hexo new (post) 新規記事タイトル
    1. マークダウンで記事を書く
    2. サーバーを立ち上げて確認
  2. hexo clean
  3. hexo deploy -g

上記の3点をコマンドプロンプトで実行するよ良い感じになる。
それぞれもうちょっと詳しく見ていく

1.hexo new (post) 新規記事タイトル

で/source/_posts/に.mdファイルが作成されるので、中身を書いていく
hexo n でも可能。
hexo new post で投稿記事、
hexo new page で固定記事ができる。

固定記事はSEOに影響しない記事を書く時などに使用する（厳密にはググって★）

サーバーを立ち上げて確認

deployする前にローカル環境で記事を確認したい場合、

または hexo s と省略可能
[http://localhost:4000](http://localhost:4000）にアクセスして確認する。

2.hexo clean

記事ができたら一度コンテンツのキャッシュクリアを行う
これを行わないとトップページの記事一覧に出ないなどの影響がある

HTMLキャッシュを削除しない場合はスキップしてもOKっぽい？

が出ればOK

3.hexo deply -g

.mdから静的ファイルを生成すし、デプロイまで一括で行うコマンド。
分解すると「hexo generate」と「hexo deploy」になる。

hexoでの記事投稿メモ

hexoで記事を投稿するためのコマンドなど

  1. hexo new (post) 新規記事タイトル
    1. マークダウンで記事を書く
    2. サーバーを立ち上げて確認
  2. hexo clean
  3. hexo deploy -g

上記の3点をコマンドプロンプトで実行するよ良い感じになる。
それぞれもうちょっと詳しく見ていく
